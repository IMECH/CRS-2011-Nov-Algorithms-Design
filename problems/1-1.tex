\begin{problem}[习题1.1]
试确定下述程序的执行步数，该函数实现一个$m\times n$矩阵与一个$n\times p$矩阵之间的乘法：

\centering
矩阵乘法运算
\begin{lstlisting}[language=C++][caption={c++}]
template<class T>
void Mult(T **a, T **b, int m, int n, int p)
{
  for(int i=0; i<m; i++)
    for(int j=0; j<p; j++){
      T sum=0;
      for(int k=0; k<n; k++)
        Sum+=a[i][k]*b[k][j];
      C[i][j]=sum;
    }
}
\end{lstlisting}
\end{problem}
